Concept Guide

The Theorem

If L is a regular language, then there exists a pumping length p such that every string w ∈ L with |w| ≥ p can be split as w = xyz satisfying three conditions.

Three Conditions

①
|y| > 0

The middle segment y must be non-empty — you must be pumping something.

②
|xy| ≤ p

The first two segments together fit within the pumping length.

③
∀i ≥ 0: xyiz ∈ L

Pumping y any number of times (including zero) must keep the string in L.

Using as a Proof

To prove L is not regular: assume it is, pick w cleverly, show that every valid split xyz produces a pumped string outside L. Contradiction!

Classic Example

L = { aⁿbⁿ | n ≥ 1 }

Any valid split must have y entirely within the a-block (since |xy| ≤ p). Pumping gives more a's than b's — not in L.
